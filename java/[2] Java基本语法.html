[2] Java基本语法

2.1 关键字和保留字

关键字：被Java语言赋予了特殊的含义，关键字中都为小写
保留字：现有Java没有使用，未来可能使用

2.2 标识符

标识符：Java对各种变量、方法和类等要素命名时使用的字符序列

> 由26个英文字母大小写，0-9，_或$组成
> 数字不能开头
> 不可以使用关键字和保留字，但能包含关键字和保留字
> 标识符不能包含空格，严格区分大小写

Java中的命名规范：
包名：多个单词全部都小写
类名、接口名：多个单词构成，所有单词首字母大写
变量名、方法名：多个单词构成，第一个单词首字母小写，第二个单词开始首字母大写
常量名：全部字母都大写，多个单词构成之间由_连接

2.3 变量

数据类型：
基本数据类型：整数类型（byte，short，int，long），浮点类型（float，double），字符型（char），布尔型（boolean）
引用数据类型：类/字符串，接口，数组

注意事项：
  • 声明long型的变量，必须要以‘l’或‘L’结尾
  • float表示的数值范围比long大
  • 字符的编码方式可能导致乱码，写入文件和读出的编码方式相同


基本数据类型转换：
自动类型转换：
当容量小的数据类型变量和容量大的数据类型变量间的运算，结果自动提升为容量大的数据类型
byte/short/char-->int-->long-->float-->double
特别的，当byte、char、short三种类型的变量做运算时，结果为int
强制类型转换：
自动类型转换的逆运算
需要使用强制转换符，并可能造成精度损失
class VariableTest3{
    public static void main(String[] args){
        // 精度损失举例 1
        double d1 = 12.3;
        //int i1 = d1; //编译错误
        int i1 = (int)d1; //强制转换
        // 精度损失举例 2
        int i2 = 128;
        byte b = (byte)i2;
        System.out.println(b); // -128
    }
}

字符串类型String
  • String属于引用数据类型
  • 声明String类型变量时，使用一对双引号
  • String可以和8种数据类型进行运算，但是只能做连接运算: +，且结果仍然是String类型
class StringTest{
    public static void main(String[] args){
        String s1 = "HelloWorld!";
        System.out.println(s1);
        String s2 = "a";
        String s3 = "";
        //char c = ''; // 编译不通过，char中有且仅有1个字符

        int number = 1001;
        String numberStr = "number:";
        String info = numberStr + number;
        System.out.println(info);
    }
}

class StringTestPractice{
    public static void main(String[] args){
        // 练习 1 ： +表示加法或者连接
        char c = 'a';
        int num = 10;
        String str = "hello";
        System.out.println(c + num + str); // 107hello
        System.out.println(c + str + num); //ahello10
        System.out.println(c + (num + str)); // a10hello
        System.out.println((c + num) + str); // 107hello
        System.out.println(str + num + c); // hello10a
        // 练习 2
        //*    *
        System.out.println("*    *"); // √
        System.out.println('*' + '\t' + '*'); // ×
        System.out.println('*' + "\t" + '*'); // √
        System.out.println('*' + '\t' + "*"); // ×
        System.out.println('*' + ('\t' + "*")); // √
    }
}

进制与进制间的转换
二进制以0B开头，八进制以0开头，十六进制以0X开头

正数的原码、反码、补码相同
负数原码，反码是原码除符号位各位取反，补码是反码加一

计算机底层都以补码方式存储数据！
